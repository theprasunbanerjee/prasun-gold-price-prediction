# 📊 Gold Price Prediction

A Python ML project exploring historical gold price data to predict future price trends.

---

## 🧠 What It Does

- Loads and cleans historical gold price datasets
- Engineers features from time-series data
- Trains a regression/classification model to forecast price direction
- Evaluates model accuracy and outputs predictions

---

## 🚀 How to Run

```bash
git clone https://github.com/theprasunbanerjee/prasun-gold-price-prediction.git
cd prasun-gold-price-prediction

pip install -r requirements.txt
# Run the notebook or main script
jupyter notebook
```

---

## 🛠️ Tech Stack

- Python 3.x
- Pandas / NumPy
- Scikit-learn
- Jupyter Notebook / Matplotlib
